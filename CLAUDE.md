# Delphine Roger — Website

## Tech Stack
- Static HTML (single page)
- Tailwind CSS via CDN (`cdn.tailwindcss.com`)
- Vanilla JavaScript (no framework, no build tools)
- Google Fonts: Playfair Display (display), Inter (body)
- No Node.js required

## Project Structure
```
website/
├── index.html          # Single-page site
├── css/style.css       # Custom styles (animations, nav, mobile)
├── js/main.js          # Interactions (nav, gallery, lightbox, form)
├── images/             # All images go here
│   ├── hero.jpg
│   ├── portrait.jpg
│   ├── costume-robe.jpg
│   ├── sur-mesure.jpg
│   ├── confection.jpg
│   └── gallery/        # Portfolio images
└── CLAUDE.md
```

## Design
- Color palette: cream (#FDFAF5), charcoal (#564D6B), lavender (#C2B0DA), sunflower (#E8B94D) — Provence pastel, light & joyful
- Inspired by farmform.be: clean, photography-forward, lots of whitespace
- Mobile-first responsive (Tailwind breakpoints + custom CSS)
- Scroll-reveal animations, lightbox gallery, filterable portfolio
